Installation

PA.js is a zero-dependency framework. You can include it in your project in several ways:

CDN

<script src="https://cdn.jsdelivr.net/npm/pa.js@latest/dist/pa.min.js"></script>

NPM

npm install pa.js

Download

Download the latest version from the download page.

Quick Start

Get started with PA.js in just a few minutes:

1. Create an HTML file

<!DOCTYPE html>
<html>
<head>
    <title>My PA.js App</title>
    <script src="https://cdn.jsdelivr.net/npm/pa.js@latest/dist/pa.min.js"></script>
</head>
<body>
    <div id="app"></div>
    <script>
        // Your PA.js code here
    </script>
</body>
</html>

2. Create your first component

const app = PA.createApp({
    data: {
        message: 'Hello, PA.js!'
    },
    template: `
        <div>
            <h1>{{message}}</h1>
            <button pa-on:click="updateMessage">Click Me</button>
        </div>
    `,
    methods: {
        updateMessage() {
            this.message = 'Button clicked!';
        }
    }
});

app.mount('#app');

Components

Components are the building blocks of PA.js applications:

Creating a Component

PA.component('my-component', {
    props: ['title'],
    data() {
        return {
            count: 0
        }
    },
    template: `
        <div class="card">
            <h3>{{title}}</h3>
            <p>Count: {{count}}</p>
            <button pa-on:click="increment">+</button>
        </div>
    `,
    methods: {
        increment() {
            this.count++;
        }
    }
});

Using a Component

<my-component title="Counter"></my-component>

State Management

PA.js provides a powerful state management system with reactive signals:

Creating a Store

const store = PA.createStore({
    state: {
        user: null,
        todos: []
    },
    mutations: {
        setUser(state, user) {
            state.user = user;
        },
        addTodo(state, todo) {
            state.todos.push(todo);
        }
    },
    actions: {
        async login({ commit }, credentials) {
            const user = await api.login(credentials);
            commit('setUser', user);
        }
    }
});

Using the Store

// In a component
export default {
    computed: {
        user() {
            return store.state.user;
        }
    },
    methods: {
        login() {
            store.dispatch('login', this.credentials);
        }
    }
}

Routing

PA.js includes a powerful routing system for single-page applications:

Defining Routes

const routes = [
    { path: '/', component: Home },
    { path: '/about', component: About },
    { path: '/users/:id', component: User }
];

const router = PA.createRouter({
    routes
});

Navigation

<!-- In template -->
<a pa-link href="/about">About</a>

<!-- Programmatically -->
router.push('/about');
